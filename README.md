Site em React + Vite + Tailwind.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Direção visual

- Tipografia: **Fraunces** (display) + **Outfit** (UI)
- Paleta: espresso, clay/terracotta, porcelain
- Layout editorial com seções imersivas, timeline de formação e procedimentos interativos
